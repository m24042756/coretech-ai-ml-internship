# Client Feedback Sentiment Analyzer

## Project Overview

This project analyzes client feedback and predicts sentiment using Machine Learning techniques.

## Features

* Data Preprocessing
* Exploratory Data Analysis (EDA)
* Sentiment Classification
* Model Comparison
* Business Insights
* Streamlit Web Application

## Models Used

1. Naive Bayes
2. Logistic Regression

## Dataset

The dataset contains:

* Feedback_ID
* Client_Name
* Service
* Feedback_Text
* Rating
* Sentiment

## Technologies Used

* Python
* Pandas
* Scikit-learn
* Matplotlib
* Streamlit

## Outcome

The project predicts whether client feedback is Positive, Neutral, or Negative and provides useful business insights.
